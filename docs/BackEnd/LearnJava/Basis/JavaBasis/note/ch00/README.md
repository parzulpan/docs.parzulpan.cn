# 编程入门

## 职业发展与提升

| 3E 模型 | 70-20-10 原则| Example|
| :-- | :-- | :-- |
| 教育（Education）| 自我学习 **10%** | 看书自学、参加培训、在线学习 |
| 曝光（Exposure）| 向他人学习 **20%** | 教授、导师、项目沟通、团队分享 |
| 经历（Experience）| 工作中学习 **70%** | 扩大工作指责、负责参与项目、发展他人 |

## 学习经验探讨

* “双核”处理，边听边思考记笔记。
* 纸上得来终觉浅，绝知此事要躬行：
  * 看得懂（视频、书籍、博客等）；
  * 练得熟（代码量要上来）；
  * 勤回顾（整理并回顾所学知识）。
